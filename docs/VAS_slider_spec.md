# iPad用 VAS スライダー設計・測定仕様

目的：VAS を iPad 11/13 上で 0–100 mm 実長に近い形で再現し、バイアスの少ない主観的疼痛評価を行うこと。

測定方法：

画面には連続した斜面とガイドのみを表示し、数値・目盛り・顔文字は表示しない。

対象者はガイドをスライドして位置を決定し、終了タップで確定候補を出す。

VAS 値を画面に表示し、対象者が確認後に再タップで記録し、ガイドは中央に戻る。

原則やり直しは不可で、必要時のみ検者が別操作で再測定。

実装要件（iPad 11/13 共通）：

PhysicalMetricsConverter などを用いて 100 mm 相当のポイント長を算出し、その幅で斜面を描画する。developer.apple+1​

SwiftUI の Slider と状態管理（@State など）でガイド位置と確定フローを実装する。
